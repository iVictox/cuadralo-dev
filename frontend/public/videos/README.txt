Coloca aquí videos cortos en formato MP4 para usar como fondo en todo el sitio web.

Archivos necesarios (3 videos):
- young-life-1.mp4: Jóvenes en fiestas, riendo, divirtiéndose
- young-life-2.mp4: Jóvenes en parques, haciendo deportes, caminando
- young-life-3.mp4: Jóvenes conversando, en cafeterías, conectando

Requisitos para cada video:
- Duración: 10-30 segundos
- Estilo: TikTok/Reels, vertical u horizontal
- Contenido: Auténtico, diverso, energético y social
- Loops suaves: El video debe poder repetirse sin que se note el corte
- Comprimir bien para web (máximo 5-10MB cada uno)

Fuentes gratuitas:
- Pexels.com/videos
- Pixabay.com/videos
- Unsplash.com (videos)

Mientras no haya videos, el sitio usa animaciones de fondo con gradientes y círculos flotantes que ya están funcionando.

¡Importante! Asegúrate de que el último frame del video sea similar al primer frame para un loop perfecto.
